# **Create Next.js app:**
1. If not, install yarn:

    ```$ npm install --global yarn```

2. Create Next app:
    
    ```$ yarn create next-app```

3. Run app:
    1. Enter app directory
        
        ```$ cd {app-directory}```

    2. Run dev
        
        ```$ yarn dev```
